# Jasmine-Renee-Houston-Photography-Catalog
This catalog allows users to browse and explore a collection of fine art photography prints. Each photo includes detailed information such as title, location, category, price, and print specifications.
The project uses an array of objects to store photo data. Each object contains:
Title
Image
Location
Category
Price
Print details (size, finish, edition)
Favorite status
Applying JavaScript methods like filter(), sort(), and find()
